## BOOTCAMP DE PYTHON | JORNADA DE DADOS
## AULA 02

Nesta aula foi estudado os tipos de dados como:
* int (inteiros);
* float (números "decimais");
* string (texto); e
* bool (valor lógico)

Dentre os tipos de dados estudados, foi desenvolvido 5 exercício de cada.

Ainda, foi possível desenvolver desafios como:

* Conversor de temperatura;
* Verificador de Palíndromo;
* Calculadora Simples;
* Classificador de Números; e
* Conversão de tipo com validação.

O interessante, é que o professor Luciano propos que realizassemos esses desafios utilizando as estruturas de TypoError, TypeCheck e Type Conversion, dando um passo a mais no desenvolvimento, e sendo possível o tratamento dos erros sem o programa crashear.

Por desafio pessoal, resolvi utilizar a estrutura de repetição _While_, que será abordada na próxima aula.

Com a utilização do _While_ foi possível tornar os desafios mais dinamicos, ou seja, quando o usuário digitava uma informação que não era esperado, o programa emitia a mensagem de erro e retornava para as opções disponíveis, sem a necessidade de rodar o código novamente.
